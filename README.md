# Obsidian 白噪音插件

一个为 Obsidian 提供白噪音播放功能的插件，帮助您专注工作和学习。

## 功能特点

- 播放多种类别的白噪音和环境声音
- 状态栏显示当前播放状态
- 点击状态栏可打开控制面板
- 支持音量调节和定时关闭功能
- 提供多种命令以便快捷控制
- 可自定义默认设置

## 使用方法

### 基本控制

1. 点击状态栏中的白噪音图标打开控制面板
2. 在控制面板中选择要播放的声音类别和具体声音
3. 使用控制面板中的按钮控制播放/暂停
4. 调整音量滑块以改变音量大小
5. 设置定时器以自动停止播放

### 命令

本插件提供以下命令，可以在 Obsidian 命令面板中使用，也可以为它们设置快捷键：

- **播放/暂停白噪音**: 切换当前白噪音的播放状态
- **停止白噪音**: 完全停止播放
- **随机播放白噪音**: 随机选择一个声音播放
- **增加音量**: 增加白噪音音量
- **减少音量**: 减少白噪音音量

### 设置

在 Obsidian 设置 > 插件选项 > 白噪音 中，您可以配置以下选项：

- **默认音量**: 调整默认播放音量
- **启动时自动播放**: 启动 Obsidian 时自动播放默认声音
- **显示状态栏**: 控制是否在状态栏显示控制图标
- **默认声音**: 选择自动播放或初始播放的默认声音
- **默认定时器**: 设置自动关闭的默认时间（分钟）

## 声音分类

该插件包含以下几类声音：

- **动物声音**: 鸟叫、猫咪呼噜、蟋蟀等自然界动物声音
- **双耳声**: 各种脑波频率的双耳节拍音频
- **自然环境**: 海浪、河流、风声等自然环境声音
- **基础噪音**: 白噪音、粉噪音、棕噪音
- **场所环境**: 咖啡馆、办公室、图书馆等场所环境声
- **雨声**: 各种雨声音效
- **物品声音**: 键盘、时钟、打字机等物品发出的声音
- **交通工具**: 火车、飞机等交通工具的声音
- **城市声音**: 城市环境中的各种声音
